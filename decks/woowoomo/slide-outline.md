## SLIDE 3 — 솔루션 개요
**제목:** 우우모: 당신이 놓친 하루를 복원하는 AI 에이전트

**한 문장 정의:**
> "우우모는 시간을 추적하는 게 아니라, 사람의 '인지 상태'를 저장하고 복원하는 시스템입니다."

**핵심 흐름:**
매일 아침 목표 설정 → 5분마다 화면 스크린샷 → Claude Vision 분석 → 목표-행동 갭 분석 → 저녁 자동 리포트 + 다음 행동 제안

**3가지 핵심 기능:**
1. **Zero-Effort Tracking** — 아무것도 입력하지 않아도 Vision AI가 자동으로 집중/딴짓 감지
2. **Contextual Time-Travel** — 중단된 작업의 맥락을 완벽 보관, 뇌의 로딩 시간 제로
3. **저녁 6시 AI 브리핑** — 매일 AI가 하루 전체를 자동 분석해 리포트 생성

## SLIDE 4 — AI 파괴력 ① 메타 레이어
**제목:** AI가 AI를 분석하는 AI

**WOW 포인트:**
> 데모: 유튜브에서 조코딩 라이브 영상을 틀어놓은 화면

- 일반 앱: "유튜브 보는 중"
- 우우모: 영상 안의 발표자 화면을 분석 → VS Code → Terminal → Slides 전환을 학습 맥락으로 감지

**2-Step Pipeline (AI 인지 편향 방어):**
- Vision 분석과 카테고리 분류를 반드시 별도 API call로 분리
- 합치면 Goodhart's Law 문제 (AI가 카테고리 맞추려고 Vision 결과 왜곡)
- AGENTS.md에 명시: "Vision + Categorization 단일 API call 절대 금지"
- → 아키텍처로 AI의 인지 편향을 방어

## SLIDE 5 — AI 파괴력 ② 저녁 브리핑
**제목:** 아무것도 안 해도, 하루가 끝나면 AI가 먼저 손을 내밉니다

**저녁 6시 자동 브리핑 — 5개 섹션:**
- 📈 **Focus Curve** — 시간대별 집중도 시각화
- 🎯 **Goal Alignment Score** — 목표 달성률 0.0~1.0
- 💡 **Tomorrow's Nudge** — AI가 제안하는 내일의 행동 3가지
- 🔧 **Bottleneck Solver** — 반복 병목 + 위임 버튼 (OpenClaw 연동)
- 🔄 **Interrupted Tasks** — 중단된 작업 + 재개 제안

## SLIDE 6 — 코딩 에이전트 깊이
**제목:** 에이전트가 에이전트를 만들었습니다 — ZERO HUMAN INTERVENTION

**① AGENTS.md: 에이전트의 헌법**
- 기술 스택 결정표, TypeScript 컨벤션 15개, Must NOT Have 가드레일, 레이어별 격리 규칙
- 에이전트가 사람에게 묻지 않고 판단하도록 사전 설계

**② Wave System: 의존성 기반 자율 실행**
- 27개 태스크 → 6개 Wave로 분리
- Wave 1–5: 병렬 자율 실행 / Wave FINAL: Plan audit · Code quality · QA · Scope check
- REJECT → 스스로 수정 후 재검증 → 4개 APPROVE 시에만 완료 보고

**③ Harness: 자기 수정 루프**
- 5시간 자율 실행, 허용된 인간 개입 단 3가지
- 각 태스크 완료 시 QA 증거 자동 저장
